What AKPsi Hopes to Acheive

Alpha Kappa Psi, hopes to give individuals all the skills they need to shape the world. Today, technology has become an increasingly large presence in both business and our lives. It almost seems as if the default consensus is "if you want to stay relevant, learn to code". We take a somewhat different approach. Not everyone needs to learn how to be a professional developer, but they should obtain a basic knowledge of the skills involved so they can stay relevant in an increasingly digitized world. One skill which we hope to share is the increasingly important ability to create websites using HTML and CSS.
